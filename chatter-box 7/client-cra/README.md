# ChatterBox Client

React frontend for the ChatterBox team communication platform.

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm start
```
Opens [http://localhost:3000](http://localhost:3000) in your browser.

### Production Build
```bash
npm run build
```

## Features
- User authentication (login/register)
- Channel browsing and creation
- Real-time message polling
- Responsive design
